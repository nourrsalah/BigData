# Kickstarter Data Analytics System

## Overview
Built an end-to-end data analytics pipeline to analyze Kickstarter projects using Python, MongoDB, and Cassandra.

## Features
- Data cleaning and preprocessing using pandas
- Feature engineering (Funding Ratio, Campaign Duration, Goal Category)
- MongoDB aggregation pipelines for analytics
- Cassandra distributed storage with batch insertion

## Technologies
- Python
- MongoDB
- Cassandra
- Pandas

## Key Insights
- Calculated project success rates across categories
- Identified top-performing campaigns
- Analyzed funding trends by country and category

## Dataset
Kaggle Kickstarter dataset (not included due to size)

## How to Run
1. Install dependencies
2. Run the Python script / notebook
